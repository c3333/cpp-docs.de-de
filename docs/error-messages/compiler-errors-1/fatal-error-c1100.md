# <a name="fatal-error-c1100"></a>Schwerwiegender Fehler C1100

OLE kann nicht initialisiert werden: System Fehlermeldung

Der Compiler kann die Component Object Model (com)-Bibliothek nicht initialisieren. Siehe [CoInitialize](/windows/win32/api/objbase/nf-objbase-coinitialize).

# <a name="compiler-error-c2122"></a>Compilerfehler C2122

"Bezeichner": Prototypparameter in der Namensliste ist ungültig.

Der-Parameter ist kein gültiger Typ. ANSI C unterstützt keine benutzerdefinierten Typen.

# <a name="expression-evaluator-error-cxx0061"></a>Ausdrucksauswertungsfehler CXX0061

Funktions Schachtelungs Tiefe überschritten

Der Ausdruck enthält eine Funktions Schachtelungs Tiefe, die den Grenzwert überschreitet. Ändern Sie den Ausdruck, um die Schachtelungs Tiefe zu verringern.

Dieser Fehler ist mit CAN0061 identisch.

# <a name="linker-tools-error-lnk1152"></a>Linkertoolfehler LNK1152

ein oder mehrere nicht ergänzte Symbole können nicht aufgelöst werden.

Diesem Fehler wird eine Warnung vorangestellt [Linkertoolwarnung LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) für jedes nicht ergänzte Symbol, das nicht aufgelöst werden konnte, und mindestens zwei Warnungen [Linkertoolwarnung LNK4002](../../error-messages/tool-errors/linker-tools-warning-lnk4002.md) für die doppelten Symbole, die für das nicht ergänzte Symbol gefunden wurden.

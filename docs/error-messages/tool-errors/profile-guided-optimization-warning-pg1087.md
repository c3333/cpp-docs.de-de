# <a name="profile-guided-optimization-warning-pg1087"></a>Warnung bei profilgesteuerter Optimierung: PG1087

Die PGC-Datei "file" ist ungültig.

Eine PGC-Datei ist beschädigt, möglicherweise aufgrund eines ungewöhnlichen Endes einer Profil Erstellungs Sitzung.  Erneutes Ausführen des Profil Erstellungs Szenarios und ordnungsgemäße Beendigung.

# <a name="resource-compiler-fatal-error-rc1009"></a>Ressourcencompiler: Schwerwiegender Fehler RC1009

Compilerlimit: Makros zu tief geschachtelt "Makro"

Die Datei überschreitet die Ressourcen Compiler-Beschränkung für die Makro Schachtelung. Überarbeiten Sie die angegebene Quelldatei, um die Schachtelungs Tiefe ihrer Makros zu verringern.

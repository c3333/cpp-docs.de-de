# <a name="project-build-error-prj0044"></a>Projektbuildfehler PRJ0044

Die Eigenschaft "zusätzliche Abhängigkeiten" für die der Datei "file" zugewiesene benutzerdefinierte Buildregel "Rule" ist ungültig. Die Eigenschaft enthält „string“, die zu „value“ ausgewertet wird.

Die Eigenschaft " **Zusätzliche Abhängigkeiten** " wurde als leere Zeichenfolge ausgewertet, oder eine Zeichenfolge, die ungültige Zeichen enthielt (ein beliebiges Zeichen, das nicht in einem Datei-oder Verzeichnisnamen enthalten sein konnte). Benutzerdefinierte Buildregeln benötigen die Ausgabe des Buildvorgangs.

Weitere Informationen finden Sie unter [Angeben von benutzerdefinierten Buildtools](../../build/specifying-custom-build-tools.md).

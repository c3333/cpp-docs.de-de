# <a name="resource-compiler-error-rc2103"></a>Ressourcencompiler: Fehler RC2103

Unerwartetes Dateiende in Zeichenfolgenliteral

Vor dem Ende einer Zeichenfolge wurde ein Dateiende gefunden. In der Zeichenfolge fehlt wahrscheinlich ein Schließ Endes doppeltes Anführungszeichen ( **"** ).

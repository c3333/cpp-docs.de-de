# <a name="compiler-error-c3501"></a>Compilerfehler C3501

Es ist kein Export der Typbibliothek für die ProgID "ProgID" registriert.

Die Klassen-ID für eine bestimmte ProgID weist keine zugeordnete Typbibliothek auf. Daher können Sie diese ProgID nicht an die `#import`-Anweisung übergeben.

# <a name="compiler-error-c2858"></a>Compilerfehler C2858

die Befehlszeilenoption '/Yc (/Fdfilename) ' ist inkonsistent mit dem vorkompilierten Header, der '/Fdfilename ' verwendet hat.

Die durch die Option vorkompilierten Header verwenden ([/Yu](../../build/reference/yu-use-precompiled-header-file.md)) angegebene Programmdatenbank ist nicht die durch die vorherige Option vorkompilierten Header erstellen ([/Yc](../../build/reference/yc-create-precompiled-header-file.md)) angegebene Programmdatenbank.

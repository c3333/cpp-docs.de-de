# <a name="compiler-error-c3603"></a>Compilerfehler C3603

' Symbol ': Typ ' Typ ' wird noch nicht unterstützt.

Sie haben versucht, einen Datentyp zu verwenden, der von der .NET-Laufzeit in einem verwalteten Objekt noch nicht unterstützt wird.

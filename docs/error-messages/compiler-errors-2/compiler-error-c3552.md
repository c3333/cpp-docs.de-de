# <a name="compiler-error-c3552"></a>Compilerfehler C3552

"Typname": Ein spät angegebener Rückgabetyp darf nicht "auto" enthalten.

Wenn Sie das- **`auto`** Schlüsselwort als Platzhalter für den Rückgabetyp einer Funktion verwenden, müssen Sie einen spät angegebenen Rückgabetyp angeben. Sie können jedoch kein anderes **`auto`** Schlüsselwort verwenden, um den spät angegebenen Rückgabetyp anzugeben. Im folgenden Codefragment wird beispielsweise der Fehler C3552 verursacht.

`auto myFunction->auto; // C3552`

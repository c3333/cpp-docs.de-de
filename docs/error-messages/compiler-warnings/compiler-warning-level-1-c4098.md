# <a name="compiler-warning-level-1-c4098"></a>Compilerwarnung (Stufe 1) C4098

"Function": void-Funktion gibt einen Wert zurück

Eine mit dem Rückgabetyp " [void](../../cpp/void-cpp.md) " deklarierte Funktion weist eine- **`return`** Anweisung auf, die einen Wert Der Compiler geht davon aus, dass die Funktion einen Wert vom Typ zurückgibt **`int`** .

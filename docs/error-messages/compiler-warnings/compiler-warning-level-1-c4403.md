# <a name="compiler-warning-level-1-c4403"></a>Compilerwarnung (Stufe 1) C4403

Unzulässiger PTR-Operator

Ein PTR-Operator wird in Inline-Assemblercode inadäquat verwendet.

Der Compiler hat den PTR-Operator ignoriert.

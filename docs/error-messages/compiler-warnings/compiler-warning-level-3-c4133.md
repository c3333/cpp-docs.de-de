# <a name="compiler-warning-level-3-c4133"></a>Compilerwarnung (Stufe 3) C4133

"Type": inkompatible Typen-von "Typ1" zu "Typ2"

Diese Warnung kann durch den Versuch verursacht werden, zwei Zeiger verschiedener Typen zu subtrahieren.

Um diese Warnung zu vermeiden, geben Sie eine geeignete Typumwandlung an.

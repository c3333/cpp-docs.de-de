# <a name="compiler-warning-level-1-c4406"></a>Compilerwarnung (Stufe 1) C4406

Operand für Direktive ignoriert

Die-Direktive übernimmt keine Operanden, aber es wurde ein Operand angegeben.

Der Compiler hat den angegebenen Operanden oder Operanden ignoriert.

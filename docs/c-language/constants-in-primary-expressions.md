# <a name="constants-in-primary-expressions"></a>Konstanten in primären Ausdrücken

Ein konstanter Operand besitzt den Wert und Typ des konstanten Werts, den er darstellt. Eine Zeichenkonstante enthält den **`int`** -Typ. Eine Integerkonstante hat einen Typ **`int`** , **`long`** , **`unsigned int`** oder **`unsigned long`** , abhängig von der Größe des Integers und von der Art, wie der Wert angegeben wird. Weitere Informationen finden Sie unter [Konstanten](../c-language/c-constants.md).
